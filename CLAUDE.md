## 项目目标
输出一个车舱场景的语音实时对话机器人（支持 rag 搜索，支持 dms，oms（照后座的鱼眼摄像头） ，和 7V 等摄像头输入，和车载中控屏屏幕读取，和利用 toolcall 控制车上座椅加热/音乐播放，和利用 gui agent 控制车载 app 进行点餐、购物、预约等服务）设计文档中文 markdown（少写代码，主要把结构和设计搞清楚），基于 OpenAI Realtime API 和 openai agents sdk。
要包含大量的习题和参考答案（答案默认折叠）。

文件组织是 index.md + chapter1.md + ...
不写代码。
提供 rule-of-thumb。

## 章节结构要求
每个章节应包含：
1. **开篇段落**：简要介绍本章内容和学习目标
2. **文字论述**：以文字论述为主，适当配上ASCII 图说明。
3. **本章小结**：总结关键概念和公式
4. **练习题**：
   - 每章包含6-8道练习题
   - 50%基础题（帮助熟悉材料）
   - 50%挑战题（包括开放性思考题）
   - 每题提供提示（Hint）
   - 答案默认折叠，不包含代码
5. **常见陷阱与错误** (Gotchas)：每章包含该主题的常见错误和调试技巧
